





:class:`split_data_sources`
===========================

.. py:class:: ansys.dpf.core.operators.logic.split_data_sources.split_data_sources(data_sources=None, output_count=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Splits a Data Sources into multiple coherent data sources, actual
   number of outputs is always less or equal to the given desired
   number of ouputs.

   :param data_sources: Data sources to split.
   :type data_sources: DataSources
   :param output_count: Number of desired outputs.
   :type output_count: int

   :returns: * **output_count** (*int*) -- Actual number of outputs.
             * **outputs1** (*DataSources*) -- Data sources outputs.
             * **outputs2** (*DataSources*) -- Data sources outputs.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.logic.split_data_sources()

   >>> # Make input connections
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)
   >>> my_output_count = int()
   >>> op.inputs.output_count.connect(my_output_count)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.logic.split_data_sources(
   ...     data_sources=my_data_sources,
   ...     output_count=my_output_count,
   ... )

   >>> # Get output data
   >>> result_output_count = op.outputs.output_count()
   >>> result_outputs1 = op.outputs.outputs1()
   >>> result_outputs2 = op.outputs.outputs2()
