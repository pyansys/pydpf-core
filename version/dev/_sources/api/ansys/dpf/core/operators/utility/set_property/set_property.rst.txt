





:class:`set_property`
=====================

.. py:class:: ansys.dpf.core.operators.utility.set_property.set_property(field=None, property_name=None, property_value=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Sets a property to an input field/field container.

   :param field:
   :type field: Field or FieldsContainer
   :param property_name: Property to set
   :type property_name: str
   :param property_value: Property to set
   :type property_value: str or int or float

   :returns: **field**
   :rtype: Field or FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.set_property()

   >>> # Make input connections
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)
   >>> my_property_name = str()
   >>> op.inputs.property_name.connect(my_property_name)
   >>> my_property_value = str()
   >>> op.inputs.property_value.connect(my_property_value)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.set_property(
   ...     field=my_field,
   ...     property_name=my_property_name,
   ...     property_value=my_property_value,
   ... )

   >>> # Get output data
   >>> result_field = op.outputs.field()
