





:class:`datasources_provider`
=============================

.. py:class:: ansys.dpf.core.operators.metadata.datasources_provider.datasources_provider(data_sources=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Creates a DataSources by expanding another.

   :param data_sources:
   :type data_sources: DataSources

   :returns: **data_sources**
   :rtype: DataSources

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.metadata.datasources_provider()

   >>> # Make input connections
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.metadata.datasources_provider(
   ...     data_sources=my_data_sources,
   ... )

   >>> # Get output data
   >>> result_data_sources = op.outputs.data_sources()
