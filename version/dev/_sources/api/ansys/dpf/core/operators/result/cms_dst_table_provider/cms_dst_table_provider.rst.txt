   Read CST table from a subfile.

   :param data_sources: Data_sources (must contain at least one
                        subfile).
   :type data_sources: DataSources

   :returns: **dst_table** -- Returns integer values of the dst table
   :rtype: PropertyField

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.result.cms_dst_table_provider()

   >>> # Make input connections
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.result.cms_dst_table_provider(
   ...     data_sources=my_data_sources,
   ... )

   >>> # Get output data
   >>> result_dst_table = op.outputs.dst_table()
