





:class:`InputsMigrateFileToVtk`

.. py:class:: ansys.dpf.core.operators.serialization.migrate_file_to_vtk.InputsMigrateFileToVtk(op: ansys.dpf.core.dpf_operator.Operator)

   Intermediate class used to connect user inputs to
   migrate_file_to_vtk operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.serialization.migrate_file_to_vtk()
   >>> my_output_filename = str()
   >>> op.inputs.output_filename.connect(my_output_filename)
   >>> my_streams_container = dpf.StreamsContainer()
   >>> op.inputs.streams_container.connect(my_streams_container)
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)
