





:class:`InputsCmsMatricesProvider`

.. py:class:: ansys.dpf.core.operators.result.cms_matrices_provider.InputsCmsMatricesProvider(op: ansys.dpf.core.dpf_operator.Operator)

   Intermediate class used to connect user inputs to
   cms_matrices_provider operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.result.cms_matrices_provider()
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)
   >>> my_matrix_form = bool()
   >>> op.inputs.matrix_form.connect(my_matrix_form)
