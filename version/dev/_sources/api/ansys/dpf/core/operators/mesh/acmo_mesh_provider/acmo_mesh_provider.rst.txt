





:class:`acmo_mesh_provider`
===========================

.. py:class:: ansys.dpf.core.operators.mesh.acmo_mesh_provider.acmo_mesh_provider(assembly_mesh=None, unit=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Converts an Assembly Mesh into a DPF Meshes container

   :param assembly_mesh:
   :type assembly_mesh: AnsDispatchHolder or Struct Iansdispatch
   :param unit:
   :type unit: str, optional

   :returns: **meshes_container**
   :rtype: MeshesContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.mesh.acmo_mesh_provider()

   >>> # Make input connections
   >>> my_assembly_mesh = dpf.AnsDispatchHolder()
   >>> op.inputs.assembly_mesh.connect(my_assembly_mesh)
   >>> my_unit = str()
   >>> op.inputs.unit.connect(my_unit)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.mesh.acmo_mesh_provider(
   ...     assembly_mesh=my_assembly_mesh,
   ...     unit=my_unit,
   ... )

   >>> # Get output data
   >>> result_meshes_container = op.outputs.meshes_container()
