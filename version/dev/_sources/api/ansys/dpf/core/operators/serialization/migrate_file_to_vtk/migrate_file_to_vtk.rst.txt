





:class:`migrate_file_to_vtk`
============================

.. py:class:: ansys.dpf.core.operators.serialization.migrate_file_to_vtk.migrate_file_to_vtk(output_filename=None, streams_container=None, data_sources=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Take an input data sources or streams and convert as much data as
   possible to vtk.

   :param output_filename:
   :type output_filename: str, optional
   :param streams_container:
   :type streams_container: StreamsContainer, optional
   :param data_sources:
   :type data_sources: DataSources, optional

   :returns: **data_sources** -- Generated output vtk file
   :rtype: DataSources

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.serialization.migrate_file_to_vtk()

   >>> # Make input connections
   >>> my_output_filename = str()
   >>> op.inputs.output_filename.connect(my_output_filename)
   >>> my_streams_container = dpf.StreamsContainer()
   >>> op.inputs.streams_container.connect(my_streams_container)
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.serialization.migrate_file_to_vtk(
   ...     output_filename=my_output_filename,
   ...     streams_container=my_streams_container,
   ...     data_sources=my_data_sources,
   ... )

   >>> # Get output data
   >>> result_data_sources = op.outputs.data_sources()
