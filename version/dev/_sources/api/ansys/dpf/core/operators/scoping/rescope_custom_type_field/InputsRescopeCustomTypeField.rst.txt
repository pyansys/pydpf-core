





:class:`InputsRescopeCustomTypeField`
=====================================

.. py:class:: ansys.dpf.core.operators.scoping.rescope_custom_type_field.InputsRescopeCustomTypeField(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   rescope_custom_type_field operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.scoping.rescope_custom_type_field()
   >>> my_fields = dpf.CustomTypeFieldsContainer()
   >>> op.inputs.fields.connect(my_fields)
   >>> my_mesh_scoping = dpf.Scoping()
   >>> op.inputs.mesh_scoping.connect(my_mesh_scoping)
   >>> my_default_value = dpf.CustomTypeField()
   >>> op.inputs.default_value.connect(my_default_value)

.. py:property:: default_value

   Allows to connect default_value input to the operator.

   :type my_default_value: CustomTypeField or CustomTypeField
