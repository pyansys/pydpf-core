





:class:`mesh_to_mc`
===================

.. py:class:: ansys.dpf.core.operators.utility.mesh_to_mc.mesh_to_mc(mesh=None, label=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Creates a meshes container containing the mesh provided on pin 0.

   :param mesh: If a meshes container is set in input, it is
                passed on as an output with the
                additional label space (if any).
   :type mesh: MeshedRegion or MeshesContainer
   :param label: Sets a label space.
   :type label: dict

   :returns: **meshes_container**
   :rtype: MeshesContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.mesh_to_mc()

   >>> # Make input connections
   >>> my_mesh = dpf.MeshedRegion()
   >>> op.inputs.mesh.connect(my_mesh)
   >>> my_label = dict()
   >>> op.inputs.label.connect(my_label)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.mesh_to_mc(
   ...     mesh=my_mesh,
   ...     label=my_label,
   ... )

   >>> # Get output data
   >>> result_meshes_container = op.outputs.meshes_container()
