





:class:`window_welch`
=====================

.. py:class:: ansys.dpf.core.operators.math.window_welch.window_welch(field=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Apply welch windowing on a given FieldsContainer having time label or
   a Field located on time. Assume that time sampling is evenly
   spaced (use time_freq_interpolation before otherwise).

   :param field:
   :type field: Field or FieldsContainer

   :returns: **field**
   :rtype: Field or FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.math.window_welch()

   >>> # Make input connections
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.math.window_welch(
   ...     field=my_field,
   ... )

   >>> # Get output data
   >>> result_field = op.outputs.field()
