





:class:`total_mass`
===================

.. py:class:: ansys.dpf.core.operators.result.total_mass.total_mass(data_sources=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Reads total mass from mode file.

   :param data_sources: Data sources (must contain at least one mode
                        file).
   :type data_sources: DataSources

   :returns: **mass** -- The unit should be grabbed from the rst file
   :rtype: float

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.result.total_mass()

   >>> # Make input connections
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.result.total_mass(
   ...     data_sources=my_data_sources,
   ... )

   >>> # Get output data
   >>> result_mass = op.outputs.mass()
