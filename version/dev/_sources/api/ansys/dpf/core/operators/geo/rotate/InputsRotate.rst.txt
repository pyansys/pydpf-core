





:class:`InputsRotate`
=====================

.. py:class:: ansys.dpf.core.operators.geo.rotate.InputsRotate(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   rotate operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.geo.rotate()
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)
   >>> my_field_rotation_matrix = dpf.Field()
   >>> op.inputs.field_rotation_matrix.connect(my_field_rotation_matrix)
