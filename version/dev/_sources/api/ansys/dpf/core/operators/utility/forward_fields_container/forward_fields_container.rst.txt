





:class:`forward_fields_container`
=================================

.. py:class:: ansys.dpf.core.operators.utility.forward_fields_container.forward_fields_container(fields=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Returns the input field or fields container.

   :param fields:
   :type fields: FieldsContainer or Field

   :returns: **fields_container**
   :rtype: FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.forward_fields_container()

   >>> # Make input connections
   >>> my_fields = dpf.FieldsContainer()
   >>> op.inputs.fields.connect(my_fields)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.forward_fields_container(
   ...     fields=my_fields,
   ... )

   >>> # Get output data
   >>> result_fields_container = op.outputs.fields_container()
