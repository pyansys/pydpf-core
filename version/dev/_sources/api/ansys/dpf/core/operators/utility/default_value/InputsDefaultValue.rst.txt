





:class:`InputsDefaultValue`
===========================

.. py:class:: ansys.dpf.core.operators.utility.default_value.InputsDefaultValue(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   default_value operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.utility.default_value()
   >>> my_forced_value = dpf.()
   >>> op.inputs.forced_value.connect(my_forced_value)
   >>> my_default_value = dpf.()
   >>> op.inputs.default_value.connect(my_default_value)
