





:class:`streams_provider`
=========================

.. py:class:: ansys.dpf.core.operators.metadata.streams_provider.streams_provider(data_sources=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Creates streams (files with cache) from the data sources.

   :param data_sources:
   :type data_sources: DataSources

   :returns: **streams_container**
   :rtype: StreamsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.metadata.streams_provider()

   >>> # Make input connections
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.metadata.streams_provider(
   ...     data_sources=my_data_sources,
   ... )

   >>> # Get output data
   >>> result_streams_container = op.outputs.streams_container()
