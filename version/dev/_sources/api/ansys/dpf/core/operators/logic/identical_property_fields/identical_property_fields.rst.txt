





:class:`identical_property_fields`
==================================

.. py:class:: ansys.dpf.core.operators.logic.identical_property_fields.identical_property_fields(property_fieldA=None, property_fieldB=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Takes two property fields and compares them.

   :param property_fieldA:
   :type property_fieldA: MeshedRegion
   :param property_fieldB:
   :type property_fieldB: MeshedRegion

   :returns: * **are_identical** (*bool*)
             * **information** (*str*)

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.logic.identical_property_fields()

   >>> # Make input connections
   >>> my_property_fieldA = dpf.MeshedRegion()
   >>> op.inputs.property_fieldA.connect(my_property_fieldA)
   >>> my_property_fieldB = dpf.MeshedRegion()
   >>> op.inputs.property_fieldB.connect(my_property_fieldB)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.logic.identical_property_fields(
   ...     property_fieldA=my_property_fieldA,
   ...     property_fieldB=my_property_fieldB,
   ... )

   >>> # Get output data
   >>> result_are_identical = op.outputs.are_identical()
   >>> result_information = op.outputs.information()
