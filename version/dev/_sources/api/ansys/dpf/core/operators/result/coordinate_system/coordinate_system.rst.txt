





:class:`coordinate_system`
==========================

.. py:class:: ansys.dpf.core.operators.result.coordinate_system.coordinate_system(cs_id=None, streams_container=None, data_sources=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Extracts the Rotation Matrix and Origin of a specific coordinate
   system.

   :param cs_id:
   :type cs_id: int
   :param streams_container:
   :type streams_container: StreamsContainer, optional
   :param data_sources:
   :type data_sources: DataSources

   :returns: **field** -- The first 9 double are the rotation (3x3
             matrix) and the last 3 is the
             translation vector
   :rtype: Field

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.result.coordinate_system()

   >>> # Make input connections
   >>> my_cs_id = int()
   >>> op.inputs.cs_id.connect(my_cs_id)
   >>> my_streams_container = dpf.StreamsContainer()
   >>> op.inputs.streams_container.connect(my_streams_container)
   >>> my_data_sources = dpf.DataSources()
   >>> op.inputs.data_sources.connect(my_data_sources)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.result.coordinate_system(
   ...     cs_id=my_cs_id,
   ...     streams_container=my_streams_container,
   ...     data_sources=my_data_sources,
   ... )

   >>> # Get output data
   >>> result_field = op.outputs.field()
