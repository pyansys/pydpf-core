





:class:`assemble_scalars_to_vectors_fc`
=======================================

.. py:class:: ansys.dpf.core.operators.utility.assemble_scalars_to_vectors_fc.assemble_scalars_to_vectors_fc(x=None, y=None, z=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Takes three scalar fields container and assembles them as a 3D vector
   fields container.

   :param x:
   :type x: FieldsContainer, optional
   :param y:
   :type y: FieldsContainer, optional
   :param z:
   :type z: FieldsContainer, optional

   :returns: **fields_container**
   :rtype: FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.assemble_scalars_to_vectors_fc()

   >>> # Make input connections
   >>> my_x = dpf.FieldsContainer()
   >>> op.inputs.x.connect(my_x)
   >>> my_y = dpf.FieldsContainer()
   >>> op.inputs.y.connect(my_y)
   >>> my_z = dpf.FieldsContainer()
   >>> op.inputs.z.connect(my_z)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.assemble_scalars_to_vectors_fc(
   ...     x=my_x,
   ...     y=my_y,
   ...     z=my_z,
   ... )

   >>> # Get output data
   >>> result_fields_container = op.outputs.fields_container()
