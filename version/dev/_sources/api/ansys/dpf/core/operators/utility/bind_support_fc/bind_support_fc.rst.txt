





:class:`bind_support_fc`
========================

.. py:class:: ansys.dpf.core.operators.utility.bind_support_fc.bind_support_fc(fields_container=None, support=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Ties a support to a fields container.

   :param fields_container:
   :type fields_container: FieldsContainer
   :param support: Meshed region or a support of the field.
   :type support: MeshedRegion or AbstractFieldSupport or TimeFreqSupport

   :returns: **fields_container**
   :rtype: FieldsContainer

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.bind_support_fc()

   >>> # Make input connections
   >>> my_fields_container = dpf.FieldsContainer()
   >>> op.inputs.fields_container.connect(my_fields_container)
   >>> my_support = dpf.MeshedRegion()
   >>> op.inputs.support.connect(my_support)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.bind_support_fc(
   ...     fields_container=my_fields_container,
   ...     support=my_support,
   ... )

   >>> # Get output data
   >>> result_fields_container = op.outputs.fields_container()
