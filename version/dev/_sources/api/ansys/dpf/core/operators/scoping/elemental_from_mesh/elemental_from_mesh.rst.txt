





:class:`elemental_from_mesh`
============================

.. py:class:: ansys.dpf.core.operators.scoping.elemental_from_mesh.elemental_from_mesh(mesh=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Retrieves the elemental scoping of a given input mesh, which contains
   the element IDs.

   :param mesh:
   :type mesh: MeshedRegion

   :returns: **mesh_scoping**
   :rtype: Scoping

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.scoping.elemental_from_mesh()

   >>> # Make input connections
   >>> my_mesh = dpf.MeshedRegion()
   >>> op.inputs.mesh.connect(my_mesh)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.scoping.elemental_from_mesh(
   ...     mesh=my_mesh,
   ... )

   >>> # Get output data
   >>> result_mesh_scoping = op.outputs.mesh_scoping()
