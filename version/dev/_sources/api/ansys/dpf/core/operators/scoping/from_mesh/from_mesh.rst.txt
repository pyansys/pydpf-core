





:class:`from_mesh`
==================

.. py:class:: ansys.dpf.core.operators.scoping.from_mesh.from_mesh(mesh=None, requested_location=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Provides the entire mesh scoping based on the requested location

   :param mesh:
   :type mesh: MeshedRegion
   :param requested_location: If nothing the operator returns the nodes
                              scoping, possible locations are:
                              nodal(default) or elemental
   :type requested_location: str, optional

   :returns: **scoping**
   :rtype: Scoping

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.scoping.from_mesh()

   >>> # Make input connections
   >>> my_mesh = dpf.MeshedRegion()
   >>> op.inputs.mesh.connect(my_mesh)
   >>> my_requested_location = str()
   >>> op.inputs.requested_location.connect(my_requested_location)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.scoping.from_mesh(
   ...     mesh=my_mesh,
   ...     requested_location=my_requested_location,
   ... )

   >>> # Get output data
   >>> result_scoping = op.outputs.scoping()
