





:class:`invariants`
===================

.. py:class:: ansys.dpf.core.operators.invariant.invariants.invariants(field=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   Computes the element-wise invariants of a tensor field.

   :param field:
   :type field: Field

   :returns: * **field_int** (*Field*) -- Stress intensity field
             * **field_eqv** (*Field*) -- Stress equivalent intensity
             * **field_max_shear** (*Field*) -- Max shear stress field

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.invariant.invariants()

   >>> # Make input connections
   >>> my_field = dpf.Field()
   >>> op.inputs.field.connect(my_field)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.invariant.invariants(
   ...     field=my_field,
   ... )

   >>> # Get output data
   >>> result_field_int = op.outputs.field_int()
   >>> result_field_eqv = op.outputs.field_eqv()
   >>> result_field_max_shear = op.outputs.field_max_shear()
