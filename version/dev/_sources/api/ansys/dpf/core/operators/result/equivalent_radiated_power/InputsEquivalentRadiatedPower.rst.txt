





:class:`InputsEquivalentRadiatedPower`
======================================

.. py:class:: ansys.dpf.core.operators.result.equivalent_radiated_power.InputsEquivalentRadiatedPower(op: ansys.dpf.core.dpf_operator.Operator)

   Intermediate class used to connect user inputs to
   equivalent_radiated_power operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.result.equivalent_radiated_power()

   >>> op.inputs.fields_container.connect(my_fields_container)
   >>> my_mesh = dpf.MeshedRegion()

   >>> my_time_scoping = int()

.. py:property:: erp_type

   Allows to connect erp_type input to the operator.

   If this pin is set to 0, the classical erp is
   computed, 1 the corrected erp is
   computed (a mesh of one face has to
   be given in the pin 1) and 2 the
   enhanced erp is computed. default is
   0.

   :param my_erp_type:
   :type my_erp_type: int

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.result.equivalent_radiated_power()
   >>> op.inputs.erp_type.connect(my_erp_type)
   >>> # or
   >>> op.inputs.erp_type(my_erp_type)

.. py:property:: boolean

   Allows to connect boolean input to the operator.

   If this pin is set to true, the erp level in
