





:class:`InputsTriMeshSkin`
==========================

.. py:class:: ansys.dpf.core.operators.mesh.tri_mesh_skin.InputsTriMeshSkin(op: ansys.dpf.core.dpf_operator.Operator)

   Bases: :py:obj:`ansys.dpf.core.inputs._Inputs`


   Intermediate class used to connect user inputs to
   tri_mesh_skin operator.

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mesh.tri_mesh_skin()
   >>> my_mesh = dpf.MeshedRegion()
   >>> op.inputs.mesh.connect(my_mesh)
   >>> my_include_surfaces = bool()
   >>> op.inputs.include_surfaces.connect(my_include_surfaces)
   >>> my_mesh_scoping = dpf.Scoping()
   >>> op.inputs.mesh_scoping.connect(my_mesh_scoping)

Property detail

.. py:property:: mesh

   Allows to connect mesh input to the operator.

   :param my_mesh:
   :type my_mesh: MeshedRegion

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mesh.tri_mesh_skin()
   >>> op.inputs.mesh.connect(my_mesh)
   >>> # or
   >>> op.inputs.mesh(my_mesh)

.. py:property:: include_surfaces

   Allows to connect include_surfaces input to the operator.

   True: meshing will also take into account
   shell and skin elements. false:
   meshing will ignore shell and skin
   elements. the default is false.

   :param my_include_surfaces:
   :type my_include_surfaces: bool

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mesh.tri_mesh_skin()
   >>> op.inputs.include_surfaces.connect(my_include_surfaces)
   >>> # or
   >>> op.inputs.include_surfaces(my_include_surfaces)

.. py:property:: mesh_scoping

   Allows to connect mesh_scoping input to the operator.

   Nodal scoping to restrict the skin extraction
   to a set of nodes. if provided, a
   skin element is added to the skin
   mesh if all its nodes are in the
   scoping.

   :param my_mesh_scoping:
   :type my_mesh_scoping: Scoping

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf
   >>> op = dpf.operators.mesh.tri_mesh_skin()
   >>> op.inputs.mesh_scoping.connect(my_mesh_scoping)
   >>> # or
   >>> op.inputs.mesh_scoping(my_mesh_scoping)
