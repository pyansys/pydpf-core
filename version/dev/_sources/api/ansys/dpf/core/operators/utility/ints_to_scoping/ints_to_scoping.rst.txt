





:class:`ints_to_scoping`
========================

.. py:class:: ansys.dpf.core.operators.utility.ints_to_scoping.ints_to_scoping(int_or_vector_int=None, location=None, config=None, server=None)

   Bases: :py:obj:`ansys.dpf.core.dpf_operator.Operator`


   take a int or a vector of int and transform it in a one entity field
   of location 'numeric'.

   :param int_or_vector_int: Int or vector of int
   :type int_or_vector_int: int
   :param location:
   :type location: str, optional

   :returns: **scoping**
   :rtype: Scoping

   .. rubric:: Examples

   >>> from ansys.dpf import core as dpf

   >>> # Instantiate operator
   >>> op = dpf.operators.utility.ints_to_scoping()

   >>> # Make input connections
   >>> my_int_or_vector_int = int()
   >>> op.inputs.int_or_vector_int.connect(my_int_or_vector_int)
   >>> my_location = str()
   >>> op.inputs.location.connect(my_location)

   >>> # Instantiate operator and connect inputs in one line
   >>> op = dpf.operators.utility.ints_to_scoping(
   ...     int_or_vector_int=my_int_or_vector_int,
   ...     location=my_location,
   ... )

   >>> # Get output data
   >>> result_scoping = op.outputs.scoping()
